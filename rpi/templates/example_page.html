Actions You Can Take

1. Check on the 'live' light profile currently running on the pond.

2. Use the Light Profile Viewer to ensure it looks good before uploading.

3. Upload and Run a new light profile and send it to the pond lights.



Excel/CSV Format

- Inlcude header: 'time' and 'intensity'

- Column 1: time (HH:MM:SS formatted as time in excel to the nearest second, can be at whatever time interval or not at intervals)

- Column 2: intensity (integer from 0 to 100, no percents!)

note: The minimum light intensity to turn Vivosun on is ~18. To be safe, assume any value < 20 turns the Vivosun 'off'.
